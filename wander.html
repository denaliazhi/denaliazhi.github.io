Wander

CS50 Final Project

Type

Programming, web dev

Tools

Python, SQL, JS

Timespan

1 week, January 2021

Wander is a gamified fitness web app that translates real world steps into in-game travel. Users set a city as their destination, walk (or run) to reach their goal, and collect a virtual souvenir for each trip completed.

Process

This project was built using the Flask web framework within CS50's IDE. For a given user, the app calls Strava's API to access their personal walking data, storing it into the SQL database. The database manages information about their travel log, current flight, and souvenir collection. Various loops and conditional statements determine the content displayed on each page.
